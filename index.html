Duas — Open Source Repository

This repository aims to publish duas and related texts for an open-source project. You can download the files below or contribute on GitHub.

About this repository

We collect, curate, and publish duas (supplications) and short Islamic texts in plain text and PDF formats. All contributions are welcome — translations, transliterations, audio timestamps, or proofreading.

How to download

  1. Click a file from the list on the right (or the download button).
  2. Files are served directly from the repository — you can also clone the repo to get everything at once.
  3. See CONTRIBUTING.md in the repo for contribution guidelines.

Contributing

Want to help? Fork the repo, add or improve files, then open a pull request. Use meaningful commit messages and include source/license information for any third-party material.
